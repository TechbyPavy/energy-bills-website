# Energy Bills - Making Electricity Bills Easy to Understand

Today, more than 70% of people in Portugal cannot understand their own electricty bill. This creates low energy literacy, worng decisions, and money wasted every month.

Energy Bill is a simple web app that helps users undertaand their bill. The user uploads the bill and receives, in few seconds a clear explanation of the values, alerts for possible mistakes, and personalized savings tips.

The prototype already shows the full flow: upload, simulated analysis, results, and suggestions; and it is prepared to include OCR and ERSE rules in the next phase. 

The goal is simples: turn a confusing bill into a userful tool taht helps people save money and make better energy choices. 

 ## Description 

This project is a prototype of a web application that allows the user to upload an electricity bill and see a simplified analysis of the main values.
Right now, the analysis is simulated, and the project is only for demonstration and learning.
